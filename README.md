# ATMOS
Explore the solar system in 3D! 
Link to try it out: https://atmos-beryl.vercel.app/

# How to Play:
- Using the mouse, you can click through which planet you want
- using the sapce bar, you can hold to rotate, or double click to automatically spin
- You can use the mouse to 'explore' the planet by hovering around it
- Thats it!
